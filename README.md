# FastApi-ReactJS

<h3>Для разработки:</h3>

#### 1. Создать виртуальное окружения:
      python3 -m venv venv
#### Активировать:
      source venv/bin/activate
<hr>

#### 2. Установить зависимости:
      pip3 install -r requirements.txt
#### Запустить проект локально &#129300;:
      uvicorn main:app --reload
#### localhost:
      http://127.0.0.1:8000/
<hr>
